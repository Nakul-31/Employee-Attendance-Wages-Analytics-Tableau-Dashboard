# 📊 Employee Attendance & Wages Analytics Dashboard

A comprehensive HR Analytics Dashboard built with **Tableau** to provide actionable insights into employee attendance, salary distribution, overtime, absenteeism, departmental performance, and workforce productivity.

---

# 📌 Project Overview

Organizations often struggle to monitor employee attendance, overtime costs, salary expenditure, and absenteeism across departments.

This project transforms raw attendance and payroll data into interactive Tableau dashboards that help HR teams and management make informed decisions.

The dashboard provides detailed visualizations for:

- Employee Attendance
- Department Performance
- Wage Analysis
- Overtime Analysis
- Salary Loss
- Workforce Productivity
- Attendance Trends
- HR KPI Monitoring
